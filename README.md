# Lemon Spire 2 / 柠檬塔2

受薄荷尖塔启发的模组，主要面向杀戮尖塔2（尤其是多人模式）提供一些便利功能。

A mod inspired by Mint Spire, mainly providing some QoL for Slay the Spire 2 (especially multiplayer mode).

WIP / 施工中

目前是我之前三个 Mod 的集合。

包含：

- 单人模式下快速 SL
- 显示即将受到的伤害
- 多人模式下显示每个玩家的伤害贡献
- （新增）显示别的玩家是否有多人专属牌（一般是仅帮助队友的效果）

A mod that combines my previous three mods.

Includes:

- Fast SL in single player mode
- Show incoming damage
- Show each player's damage contribution in multiplayer mode
- (New) Show whether other players have multiplayer-exclusive cards (generally effects that only help teammates)
